/loan/create_loan post

post

新建一个贷款项目(新建贷款第一步，然后新建联系人)

post /loan/create_loan

新建一个贷款项目(新建贷款第一步，然后新建联系人)

  • Request
  • Response

Query Parameters

  • loanType: (string )

    Example:

    qcdy(通过字典表接口获取)
  • amount: (string )

    Example:

    10000
  • spanMonth: (string )
  • startDate: (string )

    合同开始日期

    Example:

    Wed Oct 29 2014 08:00:00 GMT+0800 (CST)
  • endDate: (string )

    合同结束日期

    Example:

    Wed Oct 29 2014 08:00:00 GMT+0800 (CST)
  • payCircle: (number )

    每X月还一次款

    Example:

    2
  • payTotalCircle: (number )

    总共还款的月数

    Example:

    20
  • interests: (number )

    百分之N(按处理后的小数传递)

    Example:

    0.05（这个是百分之五）
  • assureCost: (number )

    Example:

    10000
  • serviceCost: (number )
  • overdueCostPercent: (number )

    千分之N(按处理后的小数传递)

    Example:

    0.001（这个是千分之一)
  • otherCost: (number )
  • keepCost: (number )
  • payWay: (string )

    还款方式,可能后期变动

    Example:

    {"debx":等额本息, "zqch":周期初还本付息,"zqmh":周期末还本付息,"qx":还全息}
  • firstPayDate: (string )

    首次还款时间

HTTP status code 200

Body

Type: application/json

Schema:

{
    "data": {
        "amount": 111111,
        "spanMonth": 10,
        "startDate": {
            "__type": "Date",
            "iso": "2015-01-09T16:00:00.000Z"
        },
        "endDate": {
            "__type": "Date",
            "iso": "2015-10-10T00:00:00.000Z"
        },
        "payCircle": 3,
        "payTotalCircle": 10,

        "serviceCost": 1000,

        "keepCost": 1000,
        "payWay": "debx",

            "__type": "Date",

        },
        "status": 0,
