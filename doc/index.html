/loan/create_loan post

post

新建一个贷款项目

post /loan/create_loan

新建一个贷款项目

  • Request
  • Response

Query Parameters

  • amount: (string )

    Example:

    10000
  • spanMonth: (string )
  • startDate: (date )

    合同开始日期

    Example:

    Wed Oct 29 2014 08:00:00 GMT+0800 (CST)
  • endDate: (date )

    合同结束日期

    Example:

    Wed Oct 29 2014 08:00:00 GMT+0800 (CST)
  • payCircle: (number )

    每X月还一次款

    Example:

    2
  • payTotalCircle: (number )

    总共还款的月数

    Example:

    20
  • interests: (number )

    百分之N(按处理后的小数传递)

    Example:

    5
  • assureCost: (number )

    Example:

    10000
  • serviceCost: (number )
  • overdueCostPercent: (number )

    千分之N(按处理后的小数传递)

    Example:

    0.001
  • otherCost: (number )
  • keepCost: (number )
  • payWay: (string )

    还款方式,可能后期变动

    Example:

    {"debx":等额本息, "zqch":周期初还本付息,"zqmh":周期末还本付息,"qx":还全息}
  • firstPayDate: (date )

    首次还款时间
